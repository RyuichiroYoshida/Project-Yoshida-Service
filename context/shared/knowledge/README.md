# 共有知識コンテキスト

`context/shared/knowledge/` には、「X とは何か」を説明する概念知識を置く。

## 対象例

- OAuth
- goroutine
- Clean Architecture
- RAG
- MCP
- vector search
- API design
- SQL transaction

## 書く内容

- 概要
- なぜ重要か
- 使う場面
- 注意点
- 関連概念
- 参照元
