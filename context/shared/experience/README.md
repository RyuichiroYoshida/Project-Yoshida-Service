# 共有経験コンテキスト

`context/shared/experience/` には、公開記事だけでは得られない個人の経験を蓄積する。

## 対象例

- 実装ミス
- デバッグで学んだこと
- PR レビューで受けた指摘
- 失敗したアプローチ
- うまくいったパターン
- プロジェクトの振り返り

## 書く内容

- 何が起きたか
- 原因は何だったか
- 次回どう判断するか
- 関連する知識カードやルール
