# 共有ドメインコンテキスト

`context/shared/domains/` には、複数プロジェクトで再利用できる技術領域ごとの前提、用語、判断基準を置く。

## 対象例

- backend
- database
- AWS
- security
- AI
- software architecture

## 書く内容

- その領域で重視する価値観
- 基本用語と前提
- 設計判断でよく使う観点
- 関連する知識カードや実践知へのリンク
