# 共有実践知コンテキスト

`context/shared/expertise/` には、「X をどう設計・実装するべきか」を説明する実践的な知識を置く。

## 対象例

- REST API design
- worker pool design
- repository pattern usage
- error handling strategy
- authentication design
- logging design

## 書く内容

- 採用する方針
- 判断基準
- 実装時の手順
- 失敗しやすい点
- 検証方法
