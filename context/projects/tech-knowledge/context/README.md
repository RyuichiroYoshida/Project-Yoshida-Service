# プロジェクト固有コンテキスト

`context/projects/tech-knowledge/context/` には、この技術知識管理システムにだけ適用する定義、ルール、知識、実践知を置く。

## 対象例

- knowledge card の定義
- 記事取り込みポリシー
- 要約フォーマット
- タグ付けポリシー
- AI エージェントの振る舞い
